# Search Engine (Skillbox final project)

Консольный локальный поисковый движок по набору текстовых файлов.  
Читает настройки из `config.json`, запросы из `requests.json`, строит инвертированный индекс и формирует ответы в `answers.json`.

Проект сделан по ТЗ финальной работы курса «Разработчик на C++ с нуля (2)».

---

## Возможности

- Индексация документов из списка `files` в `config.json`
- Инвертированный индекс (частотный словарь: слово → список `{doc_id, count}`)
- Поиск документов, где встречаются **все** слова запроса
- Ранжирование по релевантности:
    - абсолютная релевантность = сумма `count` всех слов запроса в документе
    - относительная релевантность = `abs / max_abs` среди найденных документов
- Ограничение количества результатов `max_responses` (по умолчанию 5)
- Многопоточная индексация документов
- Модульные тесты на GoogleTest

## Стек

- **C++17**
- **CMake**
- **nlohmann/json** (работа с JSON)
- **GoogleTest** (юнит-тесты)

---

## Структура файлов

- `config.json` — конфигурация движка и список файлов
- `requests.json` — поисковые запросы
- `answers.json` — результаты (создаётся/перезаписывается)

Пример `config.json`:
```json
{
  "config": {
    "name": "SkillboxSearchEngine",
    "version": "0.1",
    "max_responses": 5
  },
  "files": [
    "resources/file001.txt",
    "resources/file002.txt"
  ]
}****
